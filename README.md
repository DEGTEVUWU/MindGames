## Description ##

These are 5 math games that run on the console. The menu will be displayed first
then, depending on the playerâ€™s choice, one of 5 games will be displayed. The player will have to answer all questions correctly to win. If the player makes a mistake, the game will notify him and will be over. Try again to win!

Good luck!

### Requirements ###

>> Having [Git installed](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)

### Installing ###

```
>> git clone git@github.com:DEGTEVUWU/java-project-61.git
>> cd java-project-61
>> make install
```

### To run, use: ###

```
>> make run-dist
```
